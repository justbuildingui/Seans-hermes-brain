# Tweets

**What goes here:** Curated tweets that informed thinking. One page per tweet (`user-slug-YYYYMMDD.md`). Linked to `people/` and `concepts/` via typed links.

**What does NOT go here:** General social-monitoring infrastructure → `media/`. A person's broader bio → `people/`.

See [RESOLVER.md](../RESOLVER.md) for the master decision tree.
